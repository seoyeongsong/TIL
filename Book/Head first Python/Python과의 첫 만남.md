### 3. List : 복잡한 data 처리하기
### List는 배열과 같다.
- list 생성 시, interpreter는 memory에 배열과 같은 data 구조체를 생성
- data는 밑에서부터 쌓이고, 첫 번째 항목은 0, 두 번째는 1, 세 번째는 2의 index가 붙음
### How to
- data를 ("")로 둘러싸서 이름을 문자열로 변환
- 각 list 항목을 (,)로 분리
- 전체 list를 ([])로 둘러싸기
- 대입 연산자 (=) 사용해 식별자에 list 대입(assign)
